I have developed an accessible web page that retrieves event data from a JSON file, allowing users to search for events by name and date. The web page aligns with WCAG (Web Content Accessibility Guidelines) standards, ensuring inclusivity for all users. I implemented semantic HTML, ARIA attributes, and proper contrast ratios to enhance accessibility.
